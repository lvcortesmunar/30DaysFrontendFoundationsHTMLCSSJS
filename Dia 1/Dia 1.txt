👉 **header → main → section → footer**

1. Crear tu entorno (10 min)

- [ ]  Crear archivo: `index.html`
- [ ]  Abrir en VS Code
- [ ]  Instalar extensión: Live Server

 2. Escribir estructura base (NO PENSAR, SOLO COPIAR)

- [ ]  Escribe esto:

<!DOCTYPE html>
<html lang="en/sp">
<head>
  <meta charset="UTF-8">
  <title>Ejemplo</title>
</head>
<body>

  <header>
    <h1>Tirulo h1</h1>
  </header>
  <main>
    <section>
      <h2>Sección 1 con h2</h2>
      <p>Contenido de la seccion 1</p>
    </section>
  </main>

  <footer>
    <p>Footer</p>
  </footer>

</body>
</html>


3. Ver resultado (5 min)

- [ ]  Click derecho → Open with Live Server
